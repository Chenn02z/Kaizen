# Temptation Bundling

source: Atomic Habits and Tiny Habits

technique: temptation bundling, make it attractive, reward pairing

lesson: Temptation bundling makes the desired behavior more attractive by pairing it with something the user already wants. Atomic Habits frames attraction as a key lever for repetition. Tiny Habits adds that the action should still be small and prompted; the reward pairing should pull the user into starting, not mask a behavior that is too large.

use_when: Use this when the user avoids exercise, chores, stretching, meal prep, or another useful habit that feels unrewarding in the moment.

avoid_when: Avoid pairing rewards that undermine the target habit, such as using a binge snack as the reward for dieting. The reward should support or at least not sabotage the aspiration.

example_application: If the user wants to run more, reserve a favorite podcast for runs and set the tiny start as putting on shoes and playing the intro.
